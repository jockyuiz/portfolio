Algorithm a : The main bistable auxetic pattern algorithm

INPUT: conical mesh K(V.E.F), offset h,  reference orientation X ∈ C|V|, starting tilted angle ψ

OUTPUT: coordinates of planar bistable auxetic patterns S and its curved configuration S’

 

1:  procedure BISTABLE_AUXETIC_PATTERN(K,h,X,ψ)

2:            α ← DIHEDRAL_ANGLE(K)

3:            Kn ← NEUTRAL_MESH(K,h)

4:            Kn'← FLIP_MESH(Kn)

5:            M ← PROJECTION_MATRIX(K, Kn, Kn')

6:            ω,s ← VERTEX_DATA(P, Kn')

7:            T ← GRAPH_SEARCH_TREE(Kn')

8:            h, p ← HINGES_SOLVE(T, Kn',α,ω,s,ψ)

9:            S,S' ← PHYSICAL_COORDINATES(h,p,M)

10:         return S,S'

11:  end procedure
